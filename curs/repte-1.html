El primer programa

Objectiu

Escriu un programa que imprimeixi exactament aquestes tres línies:

Hola, món!
Això és JavaScript.
M'encanta programar!

El programa ha d'imprimir exactament aquest text — ni una lletra més, ni una lletra menys. Cada línia és una crida a console.log() separada.

Simulador

Escriu el teu codi i prem ▶ Executa. El sistema verificarà automàticament si la sortida és correcta.

💡 Pista

Necessites tres crides a console.log(), una per cada línia. Recorda posar el text entre cometes dobles: console.log("text");. Per l'apòstrof de «M'encanta», pots usar cometes dobles per embolcallar el text.
